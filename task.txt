Feladat:
Készíts egy standalone Java alkalmazást, mely egy egyszerű ETL (Extract, Transform, Load) rendszert valósít meg.
Az alkalmazás a megadott CSV fájlból adatokat nyer ki, átalakítja, majd betölti egy adatbázisba.
A CSV fájl emberek adatait tartalmazza, melyből megadott kritériumok alapján ki kell választani egy pár elemet, majd a kiválasztott elemeket be kell szúrni egy adatbázis táblába.

Az alábbi kritériumoknak megfelelő rekordok kerüljenek be az adatbázisba:
•	18 évnél idősebb
•	Budapesten született
•	foglalkozása: programozó

Adatbázis tábla szerkezete:
PEOPLE(ID, FIRST_NAME, LAST_NAME, AGE, PLACE_OF_BIRTH, PROFESSION)

Használandó technológiák:
-	Java 7 (preferáltabb) vagy 8
-	Spring 3 (preferáltabb) vagy 4
-	Buildelésre, dependency managementre lehetőleg Maven
-	Oracle RDBMS vagy MySQL, de használható in-memory DB is (pl. H2 vagy HSQLDB, stb.)


Nice to have feature:
Készíts egy egyszerű Java + Spring alapú web alkalmazást, ami megjeleníti az adatbázisban tárolt adatokat
